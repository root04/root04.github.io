$ heroku keys:add

you would have to watchout since it uploads ~/.ssh/id_rsa.pub in default(wo an argument).
you could specify a key file as an argument.
also, be sure to send the file ending with .pub . never send a private key file to others.

hit

$ heroku keys

in order to show the registered keys.
