Just had a hard time installing iam-fox on Windows7.

It's introduced below.

https://github.com/cookpad/iam-fox

1. clone it
2. install Cygwin
3. install createinstall( http://www.createinstall.com/) version 4.14.5
4. install xulrunner(http://ftp.mozilla.org/pub/mozilla.org/xulrunner/releases/1.9.2.19/runtimes/) version 1.9.2.19
5. place xulrunner in /usr/local/xulrunner
6. make sure the .ci file is associated to createinstall
7. boot cygwin(following operations are in cygwin)
8. execute pack-win.sh
9. if there's some kind of error, look into pack-win.sh and execute the command one by one
10. if successful, you'll get install file, so execute, and install it
11. execute iam-fox.exe !

That should be it.
