I wanted to look into apk file for it's resource.

What I did below.

0. install android-sdk
   download sdk  and install(set path)

1. install apktool
   download apktool  and place it somewhere in path. (or set a path to where it's placed)

2. execute apktool 

    

      $ apktool d

3. look through res directory
  I wanted to see the localization files, so I looked into res/value-XX folders
