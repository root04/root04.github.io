I wanted to omit specifing a user when I ssh to particular host.

$ man ssh_config

told me to edit ~/.ssh/config like below.

$ vi ~/.ssh/config

Host 192.168.122.*
   User hogehoge
Host


$ ssh 192.168.122.101
hogehoge@192.168.122.101's password:

That's it.
