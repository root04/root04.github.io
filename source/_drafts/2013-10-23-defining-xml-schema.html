When using XML as data format, if any personnel were to edit that file,

it is wise to define the XML schema.

(If some kind of application was the only thing that edits the file, the case may not be the same.


XML schema ?

XML schema defines the detailed structure of the document.

ex)
1. foo is the top element
2. foo has bar and foobar as a child
3. bar is optional
4. foobar is required

Something like above in detail.
